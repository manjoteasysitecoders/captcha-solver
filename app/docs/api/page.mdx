#  CAPTCHA APIs Overview

A unified set of APIs to solve Image, Text, and Voice CAPTCHAs using AI-powered OCR, reasoning, and speech-to-text pipelines.

---

## Authentication

All endpoints require a valid API key.

```
"x-api-key": YOUR_API_KEY
```

If the API key is missing, invalid, or revoked, requests will be rejected.

**Note -** Never expose your API key in client-side code or public repositories.

## Credit system

|  API Type	      | Endpoint         	                             | Cost per Request |
|-----------------|---------------------------------------------------|------------------|
|  Image CAPTCHA  | [/api/public/imageCaptcha](/docs/api/image-captcha) | 10 credits       |
|  Text CAPTCHA   | [/api/public/textCaptcha](/docs/api/text-captcha)	 | 10 credits        | 
|  Voice CAPTCHA  | [/api/public/voiceCaptcha](/docs/api/voice-captcha) | 10 credits       |

- Credits are deducted before processing

- Requests fail if the user has insufficient credits

- Each successful request increments totalRequests

## Image CAPTCHA API

Extract all readable text from image-based CAPTCHAs.

- Endpoint: ``POST /api/public/imageCaptcha``

- Input: Image URL (JSON)

- Output: Extracted text with original casing and spacing

See **[Image CAPTCHA API](/docs/api/image-captcha)** Documentation for full details.


## Text CAPTCHA API

Extract the final answer from text-based challenges.

- Endpoint: ``POST /api/public/textCaptcha``

- Input: Raw text (JSON)

- Output: Final answer in strict JSON

See **[Text CAPTCHA API](/docs/api/text-captcha)** Documentation for full details.

## Voice CAPTCHA API

Transcribe audio CAPTCHAs into clean, readable text.

- Endpoint: ``POST /api/public/voiceCaptcha``

- Input: Audio file or audio URL

- Output: Normalized transcription

See **[Voice CAPTCHA API](/docs/api/voice-captcha)** Documentation for full details.

## Common Error Codes

| Status |	Meaning                    |
|--------|-----------------------------|
| 400	 |  Invalid or missing input   |
| 401	 |  Missing / invalid API key  |
| 403	 |  Insufficient credits       |
| 500	 |  Internal server error      |

## Best Practices

- Always check credit balance before high-volume usage
- Prefer header-based API keys for security
- Handle pending task states gracefully
- Use HTTPS for all requests
- Cache results when possible to reduce credit usage
